개인정보 처리방침

모임 일정은 모임 생성 및 일정 조율 기능을 제공하기 위해 필요한 최소한의 개인정보를 처리합니다.

수집하는 개인정보

  • 관리자 Google 로그인 정보: Firebase Authentication을 통해 제공되는 이름, 이메일 주소, 사용자 식별자(uid)
  • 참여자 입력 정보: 참여자가 직접 입력한 이름
  • 일정 선택 데이터: 선택한 날짜, 시간, 모임 식별자, 저장 또는 수정 시각

개인정보의 이용 목적

  • 관리자 로그인 및 모임 생성자 확인
  • 모임별 참여자 식별 및 일정 선택 결과 표시
  • 초대 링크를 통한 일정 조율 기능 제공

개인정보의 저장 및 보관

로그인 정보는 Firebase Authentication에서 관리되며, 모임 정보와 일정 선택 데이터는 Cloud Firestore의 rooms/{roomId} 문서 및 하위 일정 데이터에 저장됩니다. 참여자의 브라우저에는 같은 모임에서 참여자를 구분하기 위한 임의 식별자가 localStorage에 저장될 수 있습니다.

개인정보의 제3자 제공

서비스 운영자는 개인정보를 외부에 판매하거나 임의로 제공하지 않습니다. 다만 Firebase Authentication과 Cloud Firestore 사용을 위해 Google Firebase 인프라를 통해 데이터가 처리될 수 있습니다.

보관 기간 및 삭제

모임 및 일정 선택 데이터는 모임 삭제 시 함께 삭제될 수 있으며, 삭제 요청이 있는 경우 운영자는 확인 후 필요한 데이터를 삭제합니다.

문의
